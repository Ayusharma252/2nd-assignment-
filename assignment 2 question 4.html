Here we will be talking about all types of html tags

  1. Document structure
    1.  <Html>: Represents the root of an HTML document.
    2.  <Head>: Contains meta-information about the HTML document.
    3.  <Title>: Sets the title of the HTML document, displayed in the browser's title bar or tab.
    4.  <Body<: Contains the content of the HTML document.
  2. Text Formatting 
    1. <p>: Defines a paragraph.
    2. <h1> to <h6> :Define headings (1 being the highest level and 6 the lowest).
    3. <strong> : Represents strong importance or importance.
    4. <em> :Represents emphasized text.
  3. Lists
    1. <ul> : Defines an unordered (bulleted) list.
    2.  <ol> :Defines an ordered (numbered) list.
    3.  <li> : Defines a list item.
  4. Link
    1. <a> :Defines a hyperlink. The href attribute specifies the URL.
  5. Image
    1. <img>: Embeds an image. The src attribute specifies the source (URL) of the image
  6. Tables
    1. <table>:Defines an HTML table.
    2. <tr> :Defines a table row.
    3. <td> : Defines a table cell (data).
    4.  <th> :Defines a table header cell.
  7. Semantics
    1. <div> : Defines a division or a section in an HTML document.
    2.  <span> :Defines an inline container.
    3. <article> : Represents a self-contained piece of content.
    4. <section> : Defines a section in an HTML document.
    5.  <header> : Represents a group of introductory or navigational aids
    6.  <footer> : Represents a footer for a section or a page.
    7.  <div>: Defines navigation links.
  8. Mutimedia
    1. <audio> : Embeds sound content.
    2. <video>: Embeds video content.
  9. Style and Scripting
    1. <Style>: Embeds CSS within an HTML document
    2. <Script>: Embeds or references an external script.
  10. Others
    1. <br> :Represents a line break.
    2. <hr> :Represents a horizontal rule (line).
