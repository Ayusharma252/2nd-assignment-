Types of tags in html

  1. < !DOCTYPE >- Defines the document type and version of HTML being used.

    2. Example- < !DOCTYPE html >

  2. < Html > The root element of an HTML page.
  3. < title >title tag is an HTML code tag that allows you to give a web page a title. This title can be found in the browser title bar, as well as in the search engine results pages
  4. < body > Contains the content of the HTML document, such as text, images, links, etc.
    1. < H1 to H6 > -Defines header levels, where <h1 > is the highest and < h6 > is the lowest.
    2. < paragraph >- Defines a paragraph
    3. < a >-Creates a hyperlink to another web page or resource
    4. < img > -Embeds an image in the HTML document.
    5. < ul >-Defines an unordered list.
    6. < ol > -Defines an ordered list.
    7. < li >-Defines a list item within <ul> or <ol>.
    8. < div >-Defines a division or a section in an HTML document. Often used for styling purposes.
    9. < Span >- Defines an inline section in an HTML document. Also used for styling.
